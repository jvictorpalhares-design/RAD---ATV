1 Catálogo Ativo Básica +

Pergunta: Quais são os nomes e códigos dos produtos que estão com o "Indicador de Produto Ativo" ligado?

SELECT
    Codigo_do_Produto,
    Nome_do_Produto
FROM PRODUTO
WHERE Indicador_de_Produto_Ativo = 'S';
-- resultado simulado --
1P001 | Caneta Esferográfica Azul
2P003 | Caderno Universitário
3P007 | Grampeador Médio

3 Histórico do Cliente Intermediária +

Pergunta: Quais foram os códigos de todos os vendedores que já atenderam um cliente específico (buscando pelo Nome do Cliente)?

SELECT DISTINCT
    NF.Codigo_do_Vendedor
FROM NOTA_FISCAL NF
INNER JOIN CLIENTE C
    ON NF.Codigo_do_Cliente = C.Codigo_do_Cliente
WHERE C.Nome_do_Cliente = 'NOME_DO_CLIENTE';
-- resultado simulado (cliente: "João Silva") --
1V002
2V005
2 rows returned
4 Top Produtos Avançada +

Pergunta: Qual produto teve a maior quantidade total vendida considerando todas as notas fiscais emitidas?

SELECT
    P.Codigo_do_Produto,
    P.Nome_do_Produto,
    SUM(INF.Quantidade_do_Produto) AS Total_Vendido
FROM PRODUTO P
INNER JOIN ITEM_DA_NOTA_FISCAL INF
    ON P.Codigo_do_Produto = INF.Codigo_do_Produto
GROUP BY
    P.Codigo_do_Produto,
    P.Nome_do_Produto
ORDER BY Total_Vendido DESC
LIMIT 1;
-- resultado simulado --
1P001 | Caneta Esferográfica Azul | 4.820 un.
1 row returned
5 Desempenho Regional Avançada +

Pergunta: Quantas notas fiscais foram emitidas por região?

SELECT
    R.Nome_da_Regiao,
    COUNT(NF.Numero_da_Nota_Fiscal) AS Quantidade_Notas
FROM REGIAO R
INNER JOIN VENDEDOR V
    ON R.Codigo_da_Regiao = V.Codigo_da_Regiao_do_Vendedor
INNER JOIN NOTA_FISCAL NF
    ON V.Codigo_do_Vendedor = NF.Codigo_do_Vendedor
GROUP BY
    R.Nome_da_Regiao
ORDER BY
    Quantidade_Notas DESC;
-- resultado simulado --
1Norte | 312 notas
2Sudeste | 287 notas
3Sul | 194 notas
4Nordeste | 158 notas
